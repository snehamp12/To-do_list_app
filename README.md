# To-Do List App

A simple and clean to-do list web app built using **HTML**, **CSS**, and **JavaScript**.  
Users can add, delete, and mark tasks as completed.

## Features
- Add new tasks
- Mark tasks as completed
- Delete tasks
- Fully responsive design
